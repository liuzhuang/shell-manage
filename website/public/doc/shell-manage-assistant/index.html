Assistant Skill

用 Agent 导入项目命令

Agent 会分析当前项目、验证启动命令，并把可运行的命令写入已经安装好的 ShellManage。

01　确认软件已安装

先启动 ShellManage，确认 ~/.shell-manage/config.yaml 已经创建。

02　发送导入指令

在当前项目的 Agent 对话窗口发送官网提供的导入指令。Agent 会读取 Markdown 文档并执行分析、验证和写入。

请阅读 <官网域名>/doc/shell-manage-assistant.md，按照文档分析当前项目，并将验证通过的启动命令导入 ShellManage。

03　启动并检查日志

回到 ShellManage，找到新命令并启动。看到实时日志后，首次上手即完成。
