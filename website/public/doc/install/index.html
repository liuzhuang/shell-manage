软件安装

下载并安装 ShellManage

ShellManage 当前面向 macOS。安装完成并首次启动后，再使用 Agent 导入项目命令。

01　下载安装包

从 GitHub Releases 获取当前公开版本，并选择与 Mac 芯片架构匹配的 DMG。

打开下载页面

02　安装应用

打开 DMG，将 ShellManage 拖入「应用程序」目录。安装前核对发布页面中的版本与文件信息。

03　首次启动

启动 ShellManage，等待应用创建 ~/.shell-manage/config.yaml。配置文件存在后即可导入项目命令。

Agent 读取版本 继续导入项目命令
