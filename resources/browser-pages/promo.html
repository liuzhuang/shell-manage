产品能力

运行开发命令，并看见运行结果。

ShellManage 在 macOS 上组织可独立运行的命令，并集中显示状态、实时日志、交互终端、内置浏览页面和设备指标。

产品定位

当前能力范围的简要说明。

ShellManage 是一个 macOS 桌面开发工具，用于运行命令并查看后台服务状态、实时日志和终端会话。

英文简介

与当前产品能力对应的英文说明。

Run development commands and see what happens.

完整英文说明

ShellManage organizes runnable commands on macOS and keeps service status, live logs, terminal sessions, built-in browsing, and local monitoring visible in one desktop workspace.

核心能力

  • 命令卡片：启动后台服务，或打开 SSH、REPL 和日志跟踪等交互终端。
  • 运行状态：集中展示 PID、实时日志、异常退出、停止和重启结果。
  • 内置浏览器：使用独立会话预览 Web 页面，不写入系统默认浏览器历史。
  • 设备监控：采集本机 CPU、内存、磁盘和网络指标。
  • AI 辅助：复制项目接入提示词；配置模型服务后分析日志。

常见用途

同时启动多个项目 查看实时日志 运行 SSH 命令 预览本地服务 观察设备指标
